What is HTML?

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. It uses tags — keywords wrapped in angle brackets — to define elements like headings, paragraphs, images, links, and more.

HTML 
<!-- Basic HTML Document Structure -->
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Tag Anatomy

Every HTML tag follows a consistent structure with an opening tag, optional attributes, content, and a closing tag.

SYNTAX
<tagname attribute="value"> content </tagname>

<!-- Example -->
<a href="https://example.com" target="_blank">Click Here</a>

List Tags

<ul> — Unordered List

Displays items with bullet points. Uses <li> for each item.

HTML
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
PREVIEW
  • HTML
  • CSS
  • JavaScript

<ol> — Ordered List

Displays items with sequential numbers or letters.

HTML
<ol>
  <li>Open browser</li>
  <li>Write HTML</li>
  <li>See output</li>
</ol>
PREVIEW
  1. Open browser
  2. Write HTML

Table Tags

Basic Table Structure

Tables use a hierarchy: <table> → <tr> (row) → <th> / <td> (header/data cell).

HTML
<table border="1">
  <thead>
    <tr>
      <th>Name</th>
      <th>Roll</th>
      <th>Marks</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Nipun</td>
      <td>32</td>
      <td>95</td>
    </tr>
    <tr>
      <td>Alex</td>
      <td>15</td>
      <td>88</td>
    </tr>
  </tbody>
</table>
PREVIEW
NameRollMarks
Nipun3295
Alex1588
Sara2291

colspan & rowspan

Merge cells across columns with colspan or across rows with rowspan.

HTML
<table border="1">
  <tr>
    <th colspan="2">Full Name</th>
  </tr>
  <tr>
    <td>First</td>
    <td>Last</td>
  </tr>
</table>

Links & Media

<a> — Anchor / Hyperlink

Creates clickable links to other pages, files, or sections.

HTML
<!-- External link -->
<a href="https://google.com"
   target="_blank">
  Visit Google
</a>

<!-- Email link -->
<a href="mailto:a@b.com">
  Send Email
</a>

<!-- Anchor (jump to section) -->
<a href="#section1">
  Jump to Section 1
</a>
PREVIEW
Visit Google ↗ Send Email ✉ Jump to Section 1 ↓

Form Tags

Complete Form Example

Forms collect user input. The <form> wraps all input elements.

HTML
<form action="/submit"
      method="post">

  <label>Name:</label>
  <input type="text"
    placeholder="Your name">

  <label>Email:</label>
  <input type="email"
    placeholder="you@email.com">

  <label>Gender:</label>
  <select>
    <option>Male</option>
    <option>Female</option>
  </select>

  <label>Message:</label>
  <textarea rows="3"></textarea>

  <button type="submit">
    Submit
  </button>
</form>
